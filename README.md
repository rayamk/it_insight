# 🖥️ IT-Insight

AI-Powered Hardware Analysis Tool built with Streamlit and Google Gemini API

## ✨ Features

- 📸 Upload images of IT hardware
- 🤖 AI-powered analysis using Google Gemini 1.5 Flash
- 📊 Structured output with:
  - Hardware Name
  - Primary Function
  - Compatibility
  - Key Features
  - Recommendations
- 🎨 Clean, responsive UI
- 🔒 Secure API key management
- ⚡ Fast and lightweight

## 🚀 Quick Start

### Prerequisites

- Python 3.8 or higher
- Google Gemini API key ([Get it here](https://aistudio.google.com/app/apikey))
